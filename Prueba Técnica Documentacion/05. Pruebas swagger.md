### 1. Automóvil con restricción (Jueves, placa termina en 7, hora pico)

Hoy es jueves, por lo que las placas terminadas en 7 y 8 tienen restricción. Si probamos a las 08:00 AM, no debería poder circular.

JSON

```
{
  "placa": "PCE-1237",
  "tipoVehiculo": "AUTOMOVIL",
  "exento": false,
  "fechaHora": "2026-02-28T08:00:00"
}
```

curl -X POST https://pico-placa-api-j39x.onrender.com/pico-placa/consultar -H "Content-Type: application/json" -d '{
  "placa": "PCE-1237",
  "tipoVehiculo": "AUTOMOVIL",
  "exento": false,
  "fechaHora": "2026-02-28T08:00:00"
}'

- **Resultado esperado:** `puedeCircular: false`
    

### 2. Automóvil sin restricción por horario (Jueves, placa termina en 7, hora libre)

Es el mismo auto del caso anterior y el mismo día, pero consultamos al mediodía (fuera del horario de pico y placa).

JSON

```
{
  "placa": "PCE-1237",
  "tipoVehiculo": "AUTOMOVIL",
  "exento": false,
  "fechaHora": "2026-02-26T12:00:00"
}
```

- **Resultado esperado:** `puedeCircular: true`
    

### 3. Motocicleta con restricción (Viernes, penúltimo dígito es 9, hora pico)

Tu lógica extrae el penúltimo carácter para las motos (`length - 2`). El viernes (27 de febrero) no circulan 9 y 0.

JSON

```
{
  "placa": "AA-009A",
  "tipoVehiculo": "MOTOCICLETA",
  "exento": false,
  "fechaHora": "2026-02-27T17:30:00"
}
```

- **Resultado esperado:** `puedeCircular: false`
    

### 4. Vehículo Exento (Viernes, placa termina en 0, hora pico)

Debería tener restricción, pero al marcar la bandera `exento` como `true`, el sistema debe dejarlo pasar.

JSON

```
{
  "placa": "PEA-1000",
  "tipoVehiculo": "AUTOMOVIL",
  "exento": true,
  "fechaHora": "2026-02-27T07:15:00"
}
```

- **Resultado esperado:** `puedeCircular: true`
